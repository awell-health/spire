# v0.42.1

Patch release: v0.42.0's linux goreleaser failed on a YAML parse error, so only darwin binaries shipped. This release fixes the YAML and republishes linux tarballs + docker images.

## Fixes

- **Fix `.goreleaser-linux.yaml` parse error** — the `before.hooks` block added in `spi-wps13` contained `": "` (colon+space) inside plain YAML scalars, which the parser treated as a flow mapping. Wrapped the two sed hooks in double quotes with escaped inner quotes; `goreleaser check` now validates clean (`spi-uvfqt`).

All v0.42.0 features and fixes are also in this release — see `releases/v0.42.0.md`.
