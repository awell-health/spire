# v0.41.3

## Fixes
- **Reverted glamour markdown rendering for inspector logs** — glamour treats
  single-newline-separated lines as one paragraph and reflowed wizard event
  logs into a wall of text with every `[wizard-xxx]` line smushed together.
  Wrong tool for this content. The other v0.41.2 log-visibility wins are
  preserved: logs still use the full inspector width, and `claude_runner`
  still writes args one-per-line so prompts are readable without reflow.
  Dropped `github.com/charmbracelet/glamour` from direct deps.
