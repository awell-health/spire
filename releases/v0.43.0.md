# v0.43.0

Chart + operator portability overhaul, plus the storage foundation for the
apprentice → wizard git-bundle transport.

## ⚠️ Breaking change

**`SpireAgent` CRD renamed to `WizardGuild`** (`spi-iaw8q`). `kubectl get
spireagents` stops working — use `kubectl get wizardguilds`. No migration
tooling: uninstall + reinstall is the expected rollout. The default guild
instance is shorter (no more `spire-agent-main`).

## Features

- **BundleStore interface + local backend + janitor** (`spi-8qsmr`) — storage
  foundation for apprentice → wizard git-bundle transport. `pkg/bundlestore`
  ships the interface + filesystem backend + a 5-minute janitor cycle (delete
  post-merge after 30m, orphan cleanup after 7d). Not yet wired into apprentice
  or wizard — downstream chain is `spi-1fugj` → `spi-rfee2` → `spi-k5vuh`.
- **`maxApprentices` layered config** (`spi-2ioyb`) — 3-layer setting
  (`spire.yaml` > `WizardGuild.spec` > formula step), default 3. Bounds
  concurrent apprentice subprocesses per wizard pod.
- **Wizard pod naming: `<agent>-wizard-<bead>`** (`spi-kh2em`) — drops the
  double `spire-agent-*` prefix; names in `kubectl get pods` are now legible.
- **Operator: delete `origin/feat/<bead>` after non-success reap** (`spi-3x7h7`)
  — stops leaving orphan remote branches after `test_failure` / sage-reject /
  fatal-error. Success path already cleaned locally; remote cleanup now
  follows the same discipline on failure.
- **Chart: `ResourceQuota` + `LimitRange`** (`spi-4b1z9`) —
  `values.quota.{enabled,requests,limits}` so a guild actually reserves
  namespace capacity instead of racing the scheduler. Default off.
  `LimitRange` sets per-pod min/max/default so a misconfigured guild can't
  starve others.

## Fixes

- **ClickHouse `QueryTraces` alias shadowing** (`spi-7jsgo`) — renamed aliases
  to avoid shadowing source columns.
- **ClickHouse contract test: ping `/default` endpoint** (`spi-9qs0g`).

## Upgrade notes

- CRD rename requires uninstall + reinstall for cluster installs. Hand-authored
  `SpireAgent` CRs must be rewritten as `WizardGuild`. Helm `agents:` values
  list keeps working — only the rendered CR kind changes.
- New chart values in this line: `quota.enabled` (new, off by default); plus
  `backup.enabled` and `dolthub.additionalUsers` carried from the v0.42.x line.
- BundleStore interface lands unused; no transport change until the
  `spi-1fugj`/`spi-rfee2`/`spi-k5vuh` chain closes in a follow-up release.
